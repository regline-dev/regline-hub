# regline-hub

포트폴리오 포털 — 카드로 작업물을 둘러보는 입구입니다.

- Production: https://regline-hub-three.vercel.app
- 로컬: `npm install` → `npm start` (Vite)

## 메뉴

| 사이드 | 역할 |
|--------|------|
| Projects | 완성·배포된 앱 카드 |
| Works | 진행 중 / 기록용 카드 |
| Links | 외부 링크·연락 |

## Works

왼쪽 카드 / 오른쪽 패널(50:50).

| 탭 | 내용 |
|----|------|
| 진행현황 | 각 프로젝트 GitHub README |
| 운영로그 | [CHANGELOG](https://github.com/regline-dev/CHANGELOG) 레포의 `{프로젝트}_CHANGELOG.md` |

카드 예: `챗봇/RAG 프로젝트`, `regline-hub`
